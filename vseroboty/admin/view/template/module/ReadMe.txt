Скопируйте все файлы в корневую папку. Затем установите и настройте модуль в панели администратора. 

Из панели администратора доступны настройки: id группы (номер отображаемый в url сообщества), ширина и высота виджета, выбор режима отображения (только название группы, новости группы, участники группы).
